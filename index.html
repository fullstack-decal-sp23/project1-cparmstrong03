Snake Project

Description: This project recreates the popular snake video game in the C language. It was written as part of a computer architecture class. The program has the functionality to save and read gamestates from files. The program dynamically creates a gamestate based on an input file and thus can have a variable gameboard and number of snakes. Goals of this project included dynamically changing memory allocation in order to efficiently handle memory management.

Gitlet Project

Description: This is the third and final project of UC berkeley's cs61b course, which is the second introductory computer science class in the series. This project is to create a mini version control system, similar to some of the most basic functionalities of git. These functional commands include: init, add, commit, log, global-log, status, rm, find, checkout, branch, rm-branch, reset, merge. They behave similarly to the corresponding commands in git.
